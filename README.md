# slackbot-example
slack bot example code


## Environment

Node.js 14.17.6

## Setup

`src/slack-bot.json`

```json
{
    "BotToken": "xoxb-[token_hash]"
}
```

### How to Get VerificationToken

슬랙 앱 세팅 `Basic Information` 페이지에 `Verification Token` 값이 필요하다

https://api.slack.com/apps/[APP-ID]/general?
